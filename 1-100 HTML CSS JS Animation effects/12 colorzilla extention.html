ColorZilla Extension

  1. Search on Google: ColorZilla Extension
  2. Install or Add it to your Chrome browser from the Chrome Web Store.
  3. Click the ColorZilla extension icon in your toolbar.
  4. Select: Page Color Picker Active
  5. Hover over any part of the page to pick and copy the color code.
